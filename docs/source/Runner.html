class Runner.Runner.Run(models=None, plot=False)[source]¶

Bases: object

The Run class is responsible for executing model training and evaluation processes. It supports multiple models and scalers.

Parameters:

  • models (list of str or None) – List of model names to use. If None, default models are used.

  • plot (bool) – Boolean flag to indicate whether to plot results.
